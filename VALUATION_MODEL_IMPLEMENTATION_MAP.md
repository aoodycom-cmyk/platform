## Can Unsupported Models Ever Be Selected?

- Canonical V9 path: **No evidence found that unsupported models can be selected.** `selectAndValueModels()` only invokes supported model functions and `validateAnalystBrainOutput()` rejects unsupported selected methods.
- Legacy path: **Unsupported model names can still appear in legacy suitability metadata** because `src/valuationWorkflow/workflow.js:suitableModelsFor()` includes P/B, Residual Income, DDM, AFFO, NAV, Dividend Yield, and Sum of the Parts for certain classifications. The legacy `selectValuationModels()` does not calculate those methods, so they are not weighted selected models, but they can still appear in legacy report suitability sections.
