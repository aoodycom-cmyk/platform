# Fair Value Policy v1.0

Composite Fair Value = confidence-adjusted weighted average of valid models.

Rules:
- Exclude invalid models.
- Reallocate weights only among valid models.
- Apply model confidence.
- Keep Morningstar and analyst consensus separate.
- Expected scenario value uses scenario probabilities.
- Exceptional value remains separate.

Required outputs:
each model value, weight, confidence, composite fair value, Conservative, Base, Optimistic, Exceptional, upside/downside, sensitivity.
